# ADR-009: API v2 как единственный источник истины для данных о юнитах

**Статус:** Active  
**Дата:** 2026-07-17  
**Контекст:** Данные о юнитах дублируются: статическая таблица в `internal/unitdata/units.go` и кэш API в таблице `units_reference`. При обновлении игры один источник может устареть.  
**Решение:** Единственный source of truth для характеристик юнитов (стоимость, хп, тип атаки, класс брони) — официальный API v2 (`/units/byVersion/`).  
  - `internal/unitdata/units.go` содержит только типы атак для эвристик — это мета-информация, которая не меняется между патчами (тип атаки юнита — часть дизайна, не баланса).  
  - Если тип атаки или брони юнита меняется — данные должны приходить из API.  
**Последствия:**  
  - Positive: нет расхождения данных.  
  - Negative: при старте без интернета справочник юнитов может быть пуст (нужен локальный кэш из API с TTL).  
**Связанные ADR:** ADR-008 (SQLite)
