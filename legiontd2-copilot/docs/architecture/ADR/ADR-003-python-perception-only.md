# ADR-003: ~~Python используется только для Perception (CV/OCR)~~ — SUPERSEDED

**Статус:** Superseded (2026-07-17)  
**Причина:** OCR/CV удалены из проекта. Python используется только в офлайн ML-контуре (обучение моделей).  
**Заменён на:** прямое использование Go для runtime-адвайзера, Python для офлайн ML-тренировки.  
**История:** Изначально предполагалось, что Python-сервис нужен для OCR/CV в runtime. После решения отказаться от OCR в пользу HudApi, Python-сервис удалён.
