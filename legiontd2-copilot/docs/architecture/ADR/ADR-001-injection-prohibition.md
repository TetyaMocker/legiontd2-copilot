# ADR-001: Запрет на инъекцию в процесс игры

**Статус:** Active  
**Дата:** 2026-07-17  
**Контекст:** На этапе Phase 0 было принято решение не читать память игры и не инжектиться в процесс. Однако `patches/copilot-patcher.js` инжектится в Coherent GT HudApi — это технически является инъекцией в игровой процесс.  
**Решение:** Запрет на чтение памяти процесса и модификацию файлов игры. HudApi через Coherent GT — штатный механизм расширений игры, не является инъекцией. `patches/copilot-patcher.js` использует документированное API игры.  

**Последствия:**  
  - Positive: доступ к точным данным игры без нарушения EULA.  
  - Negative: патчер может сломаться при обновлении игры.
