# ADR-012: Эвристический Advisor до накопления датасета

**Статус:** Active  
**Дата:** 2026-07-17  
**Контекст:** Без достаточного объёма данных ML-модель даёт непредсказуемые и необъяснимые рекомендации (AI2).  
**Решение:** Advisor стартует как эвристический движок на основе правил (armor/attack effectiveness, wave data). ML-модель подключается в Фазе 3, когда накоплено достаточно логов и данных API.  
  - Все рекомендации сопровождаются текстовым объяснением (FR8)  
  - Эвристики используют данные из `internal/unitdata/` (типы атак) и `internal/wavedata/` (данные волн)  
**Последствия:**  
  - Positive: объяснимые рекомендации с первого дня; возможность отладки правил.  
  - Negative: эвристики не учитывают индивидуальный стиль игрока; субоптимальны на высоких рангах.  
**Связанные ADR:** ADR-009 (API as truth)
