# ADR-007: Отказ от Overwolf в MVP

**Статус:** Active  
**Дата:** 2026-07-17  
**Контекст:** Overwolf GEP — приоритетный источник данных по ТЗ (A2), но на момент старта Phase 0 он не проверен для Legion TD 2 (открытый пункт 0.5).  
**Решение:** Overwolf не является частью MVP. Архитектура проектируется так, чтобы Overwolf можно было подключить как upgrade позже, без изменения core-компонентов.  
**Последствия:**  
  - Positive: не блокируем старт разработки неизвестным фактором.  
  - Negative: при появлении Overwolf-поддержки придётся рефакторить источник данных.  
**Архитектурная закладка:** Не требуется. Единственный источник — HudApi.  
**Связанные ADR:** ADR-010 (superseded)
