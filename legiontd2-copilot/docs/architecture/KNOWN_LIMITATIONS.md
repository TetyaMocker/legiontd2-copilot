# KNOWN LIMITATIONS

1. **Единственный источник данных**: `copilot-patcher.js` через HudApi. Если HudApi изменится или сломается, приложение потеряет данные.
2. **Нет Desktop UI**: временно используется браузерный UI. Wails не реализован.
3. **Нет тестов**: нулевое покрытие.
4. **Wave data статична**: `internal/wavedata/waves.go` может устареть после патча баланса.
5. **Датасет ограничен 3000 игр**: может иметь смещение выборки. API rate limit 1000 req/day.
6. **ML-модель не выбрана**: Phase 2 только начинается.
7. **Только Windows**: Linux/macOS не поддерживаются.
8. **Только 1920x1080 (в браузерном UI)**: координаты CSS не проверены на других разрешениях.
9. **scoreboardInfo только своей команды**: в practice/coop режиме приходят 2 игрока вместо 4. Оппоненты недоступны.
10. **Board-юниты без damageType**: field-юниты анонимны, coverage считает только hand (доступные в руке).
11. **Supply cost для fighters = 0**: стоимость призыва не экспозится в dashboard events, только у mercs.
12. **Grid в scoreboardInfo пуст без Tab**: данные поля приходят только когда зажат Tab (on-demand).
